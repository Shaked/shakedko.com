## Markdown Features

Jekyll uses Markdown for writing posts, which makes it easy to format text. Here are some examples:

### Lists

You can create unordered lists:
- Item one
- Item two
- Item three

Or ordered lists:
1. First item
2. Second item
3. Third item

### Code

You can include inline `code` or code blocks:

```python
def hello_world():
    print("Hello, World!")
    return True
```

### Links and Images

You can add [links](https://github.com) and images using standard Markdown syntax.

### Blockquotes

> This is a blockquote. It's useful for highlighting important information or quoting others.
